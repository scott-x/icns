# icns

A tool written with golang used for fast generating icns for your Mac OSX Application. Click [here](https://icon-icons.com/icon/Spider/109115) to get more icon.

### how to use?

```
# 1. install
go install github.com/scott-x/icns
# 2. run
icns
```
